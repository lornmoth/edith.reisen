Create Virutal Web Server, 1 year free Meguca Setup
  1. Launch Debian Webserver
    1. Create AWS Account
    2. Compute -> EC2
    3. Search Debian, get latest
    4. Select T2 Micro Free Tier Eligible
    5. Default settings are fine, select Launch Instance
  2. Setup Server
    1. Name & Download .pem key. Store somewhere Safe, provides root access to AWS Systems
    2. Select View Instances
      3. Copy Public DNS Open terminal Move KEY.pem to ~/.ssh Make permissions secure: chmod 0400 ~/.ssh/KEY.pem ssh into server as admin, providing key for verification: ssh admin@[public DNS] -i ~/.ssh/KEY.pem Create alias shortcut to make logging in faster: vim ~/.bashrc i to enter insert mode alias [alias_name]='ssh admin@[PUBLIC DNS]' :wq to save and quit . ~/.bashrc You can now type [alias_name] in terminal to log in directly Give your Computer SSH key access: open ~/.ssh/id_rsa.pub copy text run [alias_name] vim ~/.ssh/authorized_keys paste below pem keygen :wq Setup Install script: Copy install section of: https://github.com/bakape/meguca/blob/master/docs/installation.md vim setup.sh i + ctrl v :wq chmod +x ~/setup.sh Run the launch script sudo .~/setup.sh Setup Update section: Copy update section vim update.sh i + ctrl v :wq chmod +x ~/setup.sh
